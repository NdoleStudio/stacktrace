# v1.0.0 Release Design

## Goal

Publish the first stable GitHub release of `github.com/NdoleStudio/stacktrace`
with concise, benefit-led notes that highlight its core features.

## Release

- Tag and title: `v1.0.0`
- Target: `main` commit `807d87a`
- State: published, not a draft or prerelease

## Release Notes

```markdown
## Stacktrace v1.0.0

The first stable release is here: compact Go stack traces with the context you need and none of the noise.

- [x] Capture context-rich traces at intentional wrapping boundaries
- [x] Choose full multiline or brief single-line formatting
- [x] Carry error codes through wrapped error chains
- [x] Work naturally with `errors.Is`, `errors.As`, and `errors.Unwrap`
- [x] Run confidently on modern Go with cross-platform CI
```
